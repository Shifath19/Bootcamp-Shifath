Exercise 3: Starting with Vite

Objective:

Explore the Vite build tool to scaffold, configure, and modularize modern web projects with JavaScript and TypeScript.

Steps Followed:

1. Starter Template (JavaScript):

  • Initialized a Vite project using the vanilla JavaScript template:
    • npm create vite@latest my-js-app -- --template vanilla
  • Rebuilt the student management dashboard from Exercise 1.
  • Loaded student data dynamically from a JSON file and displayed it in a table.

2. Starter Template (TypeScript):

  • Initialized a Vite project with the TypeScript template:
    • npm create vite@latest my-ts-app -- --template vanilla-ts
  • Enhanced the dashboard:
    • Fetched data from a CSV file using PapaParse.
    • Displayed it using AG Grid with modularized code for data loading, parsing, and rendering.

3. SCSS with Vite:

  • Defined a CSS file with classes m-1, m-2, ..., m-8 where m-n = n * 2px.
  • Recreated the same in SCSS using loops.
  • Integrated SCSS into the Vite project, demonstrating its support for SCSS files seamlessly.

Challenges Faced:

  • Configuring SCSS loops and ensuring syntax correctness.
  • Managing dynamic imports in the TypeScript version.
  • Understanding and setting up AG Grid for advanced table rendering.

Outputs:

  • Two working Vite-based projects (JavaScript and TypeScript).
  • Functional dashboards with dynamic data loading and modularized code.
  • SCSS-integrated projects with flexible styling via loops.

Experience:

This exercise provided hands-on exposure to Vite's capabilities. It streamlined the development lifecycle and showcased modern tooling for modular, scalable web projects. Vite's simplicity and speed made working with SCSS and TypeScript enjoyable and efficient.
