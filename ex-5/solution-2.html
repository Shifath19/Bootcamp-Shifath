Handling Overflow Content


Overflow Handling: In .main-content, we use overflow-y: auto; to handle any overflow content vertically. This means if the content exceeds the defined height, a scrollbar will appear. The max-height: 80vh; property ensures the main content area does not take up more than 80% of the viewport height, keeping the layout balanced.


Responsive Design:


Media Query (max-width: 768px): When the screen width is smaller than 768px (tablet or mobile devices), the grid switches to a single column layout by using grid-template-columns: 1fr; and stacking the elements vertically. The grid-template-areas is adjusted to display elements one by one (header, main-content, sidebar-left, sidebar-right, footer). Media Query (max-width: 480px): For very small screens (such as phones), font sizes are adjusted using font-size to make the text more readable. The header is slightly reduced in size, and the main content font size is also adjusted.
