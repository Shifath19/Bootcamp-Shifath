Project Notes

Thought Process

The goal of this project was to create a user-friendly application for searching and filtering academic papers. The design focused on providing a clean interface while ensuring that users could easily apply various filters to narrow down their search results. The use of React allowed for a dynamic and responsive user experience.

Challenges Faced

  • Implementing a robust filtering system that could handle multiple criteria without performance issues.
  • Ensuring that the UI remained intuitive while adding more filter options like title and abstract.
  • Maintaining a consistent design across different components while allowing for customization.
  • Debugging issues related to state management in React, especially when dealing with multiple filters.

Functionalities Implemented

  • Search functionality across authors, titles, and abstracts.
  • Dynamic filtering based on user input with real-time updates to the displayed results.
  • Pagination to manage large sets of results effectively.
  • Export functionality to download filtered results in JSON format.
  • Responsive design to ensure usability on various devices.

LLMs Used
